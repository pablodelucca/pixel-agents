# OpenClaw Pixel Agents

Pixel art office where your AI agents come to life as animated characters.

## Features

- 🎨 Beautiful pixel art office environment
- 🤖 Visual representation of AI agents as animated characters
- 📊 Real-time tool usage visualization
- 🔔 Audio notifications when agents need attention
- ✏️ Built-in layout editor

## Installation

```bash
# Install dependencies
npm install
npm run install:all

# Development
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Architecture

- `webview-ui/` - React/Vite frontend
